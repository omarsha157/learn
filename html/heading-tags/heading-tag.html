heading tags

there h1,h2,h3,h4,h5,h6 heading tags, you can write html code as

<h1>heading</h1>
<h2>heading</h2>
<h3>heading</h3>
<h4>heading</h4>
<h5>heading</h5>
<h6>heading</h6>

h1 gives biggest heading use this for main heading, the following headings decrease in size from h1 to h6 use them according to importance of heading
